
## 📁 FileFlux - 简易文件传输服务器

### 🚀 项目简介

**FileFlux** 是一个基于 Python 的轻量级文件传输服务器，旨在简化文件的上传、下载和共享过程。它特别适用于以下场景：

- **开发环境文件传输**：在工作环境中，特别是在需要通过跳板机连接到开发机时，FileFlux 可以方便地进行文件的上传和下载。
- **云服务器文件管理**：在使用阿里云、百度云等云服务器进行开发时，可以通过 FileFlux 实现文件的便捷管理。
- **大文件分享**：对于微信、QQ 等平台无法直接传输的大文件，可以通过 FileFlux 生成下载链接进行分享。


### 📋 主要功能（更新版）

- **文件上传**：支持通过 Web 界面上传文件到服务器，支持中文文件名、特殊字符文件名上传，并显示上传进度条。
- **文件下载**：提供文件列表，用户可点击下载，支持中文文件名正常显示与安全编码下载。
- **文件列表展示**：自动刷新文件列表，实时显示已上传文件，支持显示原始中文文件名。
- **文件删除功能**：每个文件右侧提供“Delete”按钮，点击后弹窗确认并从服务器删除文件。
- **文件详情查看功能**：点击文件名弹出模态框，展示文件名、大小、最后修改时间，并提供一键下载入口。
- **简洁美观的界面**：采用现代化前端设计风格，使用模态框替代原生提示，提升交互体验。


### 📥 安装与部署

#### 1. **克隆项目**

```bash
git clone <your-repo-url>
cd FileFlux
```

#### 2. **安装依赖**

本项目仅依赖 Python3 标准库，无需额外安装第三方包。

#### 3. **配置项目**

- **配置文件**：项目根目录下的 `config.json` 文件用于配置服务器端口和上传目录。

  示例配置：
  ```json
  {
      "port": 8000,
      "upload_dir": "uploads"
  }
  ```


#### 4. **启动服务**

在项目根目录下运行以下命令启动服务器：

```bash
python3 server.py
```

默认情况下，服务器将在 `http://localhost:8000` 上运行。


### 🖥️ 使用说明

#### 1. **访问主页**

打开浏览器，访问 `http://localhost:8000`，即可看到 FileFlux 的主界面。

#### 2. **上传文件**

- 点击“选择文件”按钮，选择要上传的文件。
- 点击“Upload”按钮，开始上传文件。
- 上传成功后，页面会显示“上传成功”提示，并在文件列表中展示该文件。

#### 3. **下载文件**

- 在文件列表中找到要下载的文件，点击文件名即可开始下载。

### ✅ v1.2 新增功能（最新更新）

- **上传进度条功能**  
  在上传过程中动态显示进度条，提升用户对上传状态的感知，优化大文件上传体验。

- **中文文件名完整支持**  
  前端与后端协同优化，实现上传、展示、下载全过程支持中文文件名，避免 URL 编码问题。

- **文件名解码展示**  
  文件列表中显示原始中文文件名，提升可读性，实际下载仍使用编码后的安全格式。

#### ✅ v1.1 新增功能

- **文件删除功能**  
  支持通过 Web 界面删除服务器上的文件。每个文件项右侧提供“Delete”按钮，点击后可弹窗确认并从服务器移除文件。

- **文件详情查看功能**  
  点击文件名会弹出模态框，展示以下信息：
  - 文件名
  - 文件大小（字节）
  - 最后修改时间  
    同时提供一键下载入口，确保用户体验流畅。

- **增强交互体验**  
  使用模态框（Modal）替代原生 `alert()`，提升视觉效果与操作友好性，同时保留原有界面风格不变。
